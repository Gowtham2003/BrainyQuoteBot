## Requirements
  - Telegram Bot Token (Using BotFather)  `Required`

## Setup Your Own Bot
```
1. Create Your Bot From Botfather and Copy The Bot Token 

2. Create New Env Variable Called "BOT_TOKEN" and Place your token in it 

3. Your Bot is Ready to Host. 
```
